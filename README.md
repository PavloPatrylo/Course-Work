# Моделювання впливу зміни клімату на ризики прибережних затоплень з використанням ГІС та сценарного підходу.

Інтерактивна веб-платформа для моделювання та аналізу ризиків затоплення прибережної зони Одеси внаслідок підняття рівня моря. Проєкт базується на кліматичних сценаріях **IPCC AR6** та використовує геоінформаційні алгоритми для точного розрахунку зон затоплення.

## Опис проєкту

Ця курсова робота присвячена розробці програмного забезпечення для оцінки вразливості території Одеси до змін клімату. Система дозволяє візуалізувати зони потенційного затоплення на основі цифрової моделі рельєфу (DEM) та розраховувати втрати території.

### Ключові можливості:
1.  **Інтерактивне моделювання:** Плавна зміна рівня моря від 0 до 5 метрів за допомогою слайдера.
2.  **Сценарії IPCC AR6:** Інтеграція офіційних прогнозів (SSP1-1.9, SSP2-4.5, SSP5-8.5) до 2150 року.
3.  **Алгоритм BFS (Breadth-First Search):** Використання алгоритму пошуку в ширину для моделювання фізичного розливу води (врахування дамб та перешкод), на відміну від простого відсікання за висотою ("наївний" метод).
4.  **Точна геодезія:** Розрахунок площі затоплення ($м^2$) з урахуванням викривлення проєкції Меркатора (косинусна поправка на широту Одеси).
5.  **Експорт даних:** Можливість завантажити результат моделювання у векторному форматі **GeoJSON** 
6.  **Аналітика:** Порівняльні діаграми та гістограми розподілу висот.
